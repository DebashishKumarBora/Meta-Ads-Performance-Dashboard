# Meta Ads Performance Dashboard 📈  

## 📌 Introduction  
This **Looker Studio** dashboard tracks performance metrics for Meta (Facebook/Instagram) advertising campaigns, analyzing:  
1. **Cost efficiency** (CPM, CPC, CTR)  
2. **Traffic quality** (Engagement rates, Session behavior)  
3. **Audience conversion patterns**  

---

## 📊 Data Overview  
### **Core Metrics Tracked**  
- **Campaign Performance**  
  - Impressions, CTR, CPC  
  - Cost per Result ($0.9 avg)  
- **Website Behavior**  
  - 429,808 sessions from Meta  
  - 134,732 engaged sessions (11.2s avg)  
- **Monthly Trends**  
  - Session growth by source  
  - Engagement rate fluctuations  

---

## 🎯 Project Objectives  
1. **Identify** top-performing campaigns by ROI  
2. **Analyze** traffic quality from Meta sources  
3. **Optimize** ad spend based on engagement data  
4. **Track** monthly performance trends  

---

## 🧹 Data Processing  
- Calculated **Engagement Rate** = (Engaged Sessions / Total Sessions)  
- Normalized **Cost Metrics** across 20+ campaigns  
- Derived **Quality Scores** for traffic sources  
- Aggregated **Monthly Trends** for YOY comparison

## 🔍 Key Findings  
- **Top Campaign**: "Italian Carnegie - Dubai" (27K impressions, $3,937 spend)  
- **Best CTR**: Flying Group plc (48% click-through rate)  
- **Engagement**: 31% avg. engagement rate from Meta traffic  
- **New Users**: 199,783 acquired via Meta ads  

---

## 🚀 Recommended Actions  
1. **Increase budget** for high-CTR campaigns (Flying Group plc + Italians)  
2. **Pause** underperformers ("Fish Trading" - 1.3% CTR)  
3. **Optimize ad creatives** for mobile (11.2s avg engagement time)  
4. **Reallocate** 15% of December budget to November (peak engagement)
